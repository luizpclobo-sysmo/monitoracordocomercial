# Padrao de Codificacao

Indice dos padroes de codificacao por linguagem/framework.
Consulte o arquivo especifico para detalhes de cada tecnologia.

## Linguagens e Frameworks

| Arquivo | Tecnologia | Descricao |
|---------|-----------|-----------|
| [java.md](java.md) | Java | Padroes e convencoes para Java |
| [java-quarkus.md](java-quarkus.md) | Java + Quarkus | Padroes especificos para projetos Quarkus |
| [react.md](react.md) | React | Padroes para projetos React/JavaScript/TypeScript |
| [delphi.md](delphi.md) | Delphi | Padroes e convencoes para Delphi/Object Pascal |
| [delphi-sysmo.md](delphi-sysmo.md) | Delphi + Sysmo | Convencoes especificas para projetos Sysmo (logs, SQL, VO, encoding) |

## Como usar

- Ao iniciar um projeto em uma dessas tecnologias, consulte o arquivo correspondente
- Atualize os padroes conforme novas decisoes forem tomadas
- Para adicionar uma nova linguagem, crie o arquivo .md e registre neste indice
